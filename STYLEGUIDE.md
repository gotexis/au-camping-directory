# Style Guide

## 设计参考
- [hipcamp.com](https://hipcamp.com) — 借鉴了卡片式布局、地图为中心的探索体验、清新自然配色
- [wikicamps.com.au](https://wikicamps.com.au) — 借鉴了设施图标系统、州级分类导航
- [recreation.gov](https://recreation.gov) — 借鉴了筛选系统、简洁的信息层级

## 配色方案
DaisyUI theme: forest (绿色自然主题，深色调配合户外氛围)

## 设计决策
- 首页：州地图 + 搜索 + 统计数据
- 州页面：该州所有camping sites列表 + 地图
- 单个site页面：详细信息 + 设施图标 + 地图标记
- 卡片风格：圆角、浅绿色边框、设施图标行
- 地图使用 Leaflet (MapView组件)，所有site可在地图上浏览
